# VirusPilot's Collision Avoidance Compilation of Information
# Basics
## HF Protocols
- 868MHz: OGN, FLARM, FANET, ADS-L, PilotAware
- 978MHz: UAT, FIS-B, TIS-B
- 1090MHz: ADS-B
## DATA Protocols
- OGN, (FLARM-)NMEA, FANET, ADS-L, GDL90
